Website Design Revision

The purpose of this website is to provide a crash course in the history of website design, how it has evolved, and where it stands today.

The main focus will be on the idea of the 'separation of concerns' -- keeping the three cores of a website (the information, the design, and the functionality) separate from each other.

Use the navigation buttons on the left hand side to move around the website. You also have the option to test your knowledge on the subject by taking a quiz.

On the right hand side, there is the option to change the look of this website. Use the COLOUR menu to change the colour scheme, or use the TEXT menu to alter the size of the text on the screen.

Each page of information will also include code snippets. These snippets will have an arrow at the top, where if you click it, you will see a live preview of what that code is showing you.
